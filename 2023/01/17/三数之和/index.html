题目

15. 三数之和

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请

你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

思考

大致思路

  • 这题很明显就是用双指针
  • 先排序（不要忘记！！！！经常忘记）
  • 然后固定住一个指针，然后在这个指针后面定义两个指针进行范围的增减

难点

  • 去重
    • eg[0,0,0]
    • 不但固定住的那个指针，还是后面定义的两个指针，都会可能有重复的出现，这样会导致多种重复的结果，所以注意去重
